# LULC-Mapping-GEE
Land Use Land Cover (LULC) classification of Hassan District using Landsat 9 and SVM in Google Earth Engin

## Dataset Used
Landsat-9 satellite imagery

## Study Area
Define the region of interest (ROI) used for the analysis.

## Methodology
1. Import satellite imagery
2. Preprocess the data
3. Define training samples
4. Apply supervised classification
5. Generate LULC map

## Platform
Google Earth Engine

## Output
LULC classification map of the study area.
